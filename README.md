# TS Command Center

A modern AI engineering portfolio built with Next.js, React, TypeScript and Tailwind CSS.

---

## Vision

TS Command Center is not a traditional portfolio.

It is an interactive dashboard that showcases my journey, projects, technical skills, achievements and future goals as an AI Engineer and future AI Founder.

---

## Tech Stack

- Next.js 16
- React 19
- TypeScript
- Tailwind CSS v4
- Framer Motion
- Lucide React
